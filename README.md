# Simple IP Blocker for Pages

A lightweight WordPress plugin that allows administrators to restrict access to selected pages based on a visitor’s IP address.  
Designed for simplicity, efficiency, and clarity — no technical complexity required.

---

## Features
- Block access to specific WordPress pages by IP address
- Support for single IPs and ranges (CIDR)
- Lightweight and fast — no impact on site performance
- Simple admin interface with clear workflows
- Multilingual support (English & French included)

---

1. Download simple-ip-blocker-for-pages.zip

  https://github.com/Pmat1000/simple-ip-blocker-for-pages/raw/refs/heads/main/simple-ip-blocker-for-pages.zip

## Method 1: Upload via WordPress Admin
1. Download the plugin ZIP file from GitHub.
2. Log in to your WordPress admin dashboard.
3. Go to **Plugins → Add New → Upload Plugin**.
4. Click **Choose File**, select the ZIP file, then click **Install Now**.
5. Once installed, click **Activate Plugin**.
